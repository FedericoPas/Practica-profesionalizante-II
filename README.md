# Aprende Mecanica Automotriz

Proyecto web para un instructor de mecanica automotriz que desea publicar, organizar y vender cursos online.

## Tecnologias

- Frontend: React + Vite
- Backend: Node.js + Express
- Publicacion sugerida: Vercel para el frontend y un servicio Node compatible para la API

## Estructura

```text
.
├── frontend/   Aplicacion React
└── backend/    API REST con Express
```

## Instalacion

```bash
npm.cmd install
```

## Ejecutar en desarrollo

En una terminal:

```bash
npm.cmd run dev:backend
```

En otra terminal:

```bash
npm.cmd run dev:frontend
```

Frontend: http://localhost:5173

Backend: http://localhost:4000/api/health

## Funcionalidades iniciales

- Listado de cursos de mecanica automotriz.
- Detalle visual de nivel, duracion, precio y modulos.
- Formulario de inscripcion.
- Formulario de contacto.
- API inicial para cursos, inscripciones y consultas.
- Diseno responsive para celulares, tablets y computadoras.
